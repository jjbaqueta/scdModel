The files present in this folder represent a DS-net build through a random network topology. The network comprises 85 agents, which can change their social behavior over time. The file DSNET_1.xml represents the first stage of the DS-net, the file DSNET_2.xml represent the second stage, where new behaviors are randomly assigned to agents, and the files DSNET_3.xml and DSNET_4.xml represent the last stage of the DS-net, both files are equals, specifying the second change of the agents' behavior.
